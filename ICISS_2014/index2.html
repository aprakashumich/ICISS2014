ICISS (International Conference on Information Systems Security), provides a forum for discussing and disseminating recent advances in information and systems security. ICISS 2014, the 10th edition of this annual conference, will be held at IDRBT, Hyderabad, India. The proceedings are traditionally published as part of the Lecture Notes in Computer Science by Springer. ICISS 2014 encourages submissions from the academia, industry and government addressing both theoretical and practical problems in information systems security and related areas. The conference will hold several keynotes by renowned scientists, panel discussions and tutorials on current topics, and PhD forums to advise doctoral students.

  • Application Security
  • Automated Security Checking
  • Automated Security Alert Systems
  • Biometric Security
  • Cloud Security
  • Cyber Security monitoring and control
  • Cryptographic Protocols
  • Cyber-physical Systems Security
  • Data Security and Privacy
  • Digital Forensics
  • Distributed Systems Security
  • End Point Security
  • Formal Models in Security
  • Identity Management
  • Intrusion Detection and Prevention
  • Intrusion Tolerance and Recovery
  • Language-based Security
  • Malware Analysis and Mitigation
  • Network Security
  • Operating Systems Security
  • Perimeter Security
  • Phishing, Pharming, Spamming, attacks and counter measures
  • Privacy and Anonymity
  • Secure Data Streams
  • Secure and Smart Banking
  • Security Monitoring, Assessment and Alerting systems
  • Security and Usability
  • Security Testing

  • Vulnerability Detection and Mitigation
  • Usable Security
  • Web Security
